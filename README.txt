LaunchCountdownClock is (planned to be) a hunk of code to run on a custom piece of hardware that I plan to build to simulate the famous countdown clock at the Kennedy Space Center complex where NASA launches (for a little longer) shuttle missions.

In the beginning, there was an engineer with a box full of stuff like and MSP430 target development board, some red/green 8x8 led matrix blocks, and a desire to do something cool with them...
